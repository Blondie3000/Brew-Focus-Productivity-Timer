## ‚ú® About Brew Focus

**Brew Focus** is a sleek, minimalist productivity timer designed to help you maintain deep work sessions using the renowned **Pomodoro Technique**. It combines the structure of focused work with the simple ritual of taking a break, ensuring you stay refreshed and efficient throughout your day.

The app's design philosophy is centered around a distraction-free experience, allowing you to focus on your task without unnecessary clutter.

## üöÄ Key Features

* **Pomodoro Cycles:** Effortlessly guides you through **25-minute work sessions** followed by scheduled breaks.
* **Customizable Timers:** Easily adjust the length of the focus and break periods to fit your personal workflow.
* **Automated Transitions:** Seamlessly switch between work and break timers without manual intervention.
* **Minimalist Design:** A clean, uncluttered interface that prioritizes focus and ease of use.
* **Audio Alerts:** Gentle notifications to indicate the end of a session without being disruptive.

## ‚öôÔ∏è How to Use

1.  **Start:** Click the **Start** button to begin your first 25-minute focus session.
2.  **Focus:** Work intensely on your task until the timer rings.
3.  **Short Break:** The timer will automatically switch to a **5-minute short break**. Use this time to stretch, grab a drink, or clear your mind.
4.  **Long Break:** After completing four focus sessions, the timer will initiate a **15-minute long break** to help you fully recharge.
5.  **Repeat:** Continue the cycle until your task is complete or your workday is over!

## üíª Tech Stack

This project is built using:

* Language: TypeScript (TSX)., HTML
* Core Framework: React (v19) (using Hooks like useState, useEffect, useRef).
* Styling: Tailwind CSS (configured via CDN in index.html with a custom coffee theme).
* Icons: Lucide React.
* AI Integration: Google GenAI SDK (@google/genai) for the Barista Bot.
* Build/Runtime: It appears to be a browser-based Single Page Application (SPA) using ES Modules via an importmap in index.html.
